An Abstract System for Computation

Abstract

This document describes the representation and interpretation of programs written in a programming language called the Abstract System for Computation, abbreviated as the name "ASC". ASC is intended to be a highly expressive, general purpose programming language. It is designed to be capable of expressing a wide range of useful algorithms.

ASC minimally requires the printable set of characters from the ASCII character encoding table. This also means that implementations of ASC can easily be extended to support Unicode character encodings.

Symbols in ASC are formed by sequences of characters and delimited by blank space and control characters.

A special type of symbol is called a constant. Constants represent numeric values and will self-evaluate to whichever value they represent. A constant is any symbol which contains all decimal characters.

A constant can be written with a + or - as the first character. This denotes a positive or negative value.
