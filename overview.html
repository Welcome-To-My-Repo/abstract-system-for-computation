Overview

This document describes the properties of an Abstract System for Computation (ASC), a general purpose tool for computing.

Scope

This document specifies...

  • the syntactic and semantic properties of the ASC language.
  • the capabilities and limitations of the ASC environment.
  • all assumptions made about the world outside the environment.
  • any data which should be provided by default
  • the methods for interpreting the ASC language

This document does not specify...

  • requirements for anything outside the environment.
  • methods for translating the ASC language to another language
  • the storage format of the environment and data in it

Features

ASC provides a session-based environment where data is held in an image. The image is an archival format which can be exported from and imported to a session. This allows data to persist between multiple sessions.

Each session allows one or more bidirectional streams for input and output to the environment. These streams are used to submit programs to the environment for computation and to recieve data resulting from any invocations of the display function.
