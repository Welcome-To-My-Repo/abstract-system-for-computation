Overview

This document describes the properties of an Abstract System for Computation, a general purpose tool for creating programs.

Scope

This document specifies...

  • the syntactic and semantic properties of the ASC language
  • all assumptions made about the capabilities of the underlying environment.
  • any algorithms which should be provided by default
  • the constructs which should be provided by default
  • the methods for interpreting the ASC language

This document does not specify...

  • requirements for the underlying environment.
  • methods for translating the ASC language to another language
