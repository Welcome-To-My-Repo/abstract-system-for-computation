Introduction

This document contains the specification of "nth", a system for creating and performing computations. This system is intended to be generally useful when applied to any problem which requires computing.

The system is comprised of a description language for composing programs and a small environment to compute those programs. In this document, the name "nth" refers to the description language. The term "environment" shall refer to the mechanisms and resources through which computation is facilitated.

As this document is a specification, all properties and behaviors of the system as described are considered requirements for an implementation.

Syntax

nth is written as a sequence of characters. The minimal set of characters required to write an nth program is supported by the printable subset of the ASCII character encoding (all characters excluding control characters). nth is intended to be compatible with any character encoding which provides the same set of printable characters as the ASCII encoding. This document uses this subset with the addition of a line break character.

For the purposes of this standard, characters are classified into the following catagories: letters, digits, blanks, and glyphs.

letters
Any character associated with a spoken phoneme.
digits
Specifically the ten decimal digits, 0123456789.
blanks
Any character which results in empty space when printed.
Glyphs
All other characters which are not control or formatting characters and display some symbol when printed.
